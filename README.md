# Simulation-MAC-Exponential-Backoff
Discrete-event simulation of a MAC protocol using exponential backoff. Models multiple stations sharing a channel, handling arrivals, collisions, retransmissions, and measuring performance.

Notre code simulateur "Mac_simulateur" se trouve dans le main sous forme python. De plus, nous fournisons notebook jupyter qui contient notre code python en blocs nous permettant de faire les simulations.
